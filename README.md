ОСКР Портал
===========
Портал для регистрации заявок на ВКС и бронирования помещений для совещаний

Переменные окружения
--------------------
- HTTPD_FRONTEND_SERVER_NAME
- HTTPD_BACKEND_SERVER_NAME
- MAILER_HOST
- MAILER_USERNAME
- MAILER_PASSWORD
- MAILER_PORT
- MYSQL_DBNAME
- MYSQL_USERNAME
- MYSQL_PASSWORD
- MONGO_DBNAME
